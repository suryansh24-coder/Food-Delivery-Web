# 📖 About The Project

**Feast Fast** is a **modern food delivery web application UI** built using **Vue.js**, **Vuetify**, **Vue Router**, **Pinia**, and **Vite**.

This project was developed as a **frontend practice assignment** to improve skills in:

- Modern Vue.js Development
- Component Based Architecture
- Responsive UI Design
- State Management
- Routing
- Clean Code Structure
- User Experience Design

> ⚠️ **Important**
>
> This is **NOT a production-ready application.**
>
> It is a **frontend-only practice project** built using **dummy/static data**.
>
> There is **no backend**, **no authentication**, **no payment gateway**, and **no real database**.

---

# 🎯 Project Objective

The objective of this project is to simulate the UI of a modern food delivery platform while learning:

- Vue.js Fundamentals
- Vuetify Components
- Responsive Layouts
- Component Reusability
- State Management using Pinia
- Routing using Vue Router
- Clean Folder Architecture
- Frontend Best Practices

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/feast-fast.git
```

---

## Navigate into Project

```bash
cd feast-fast
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

---

## Build Production

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```
